# Code of conduct

Graphward is committed to a welcoming, respectful, and harassment-free community.

## Expected behavior

- Be considerate, constructive, and specific.
- Critique ideas and code without attacking people.
- Respect differing experience levels, backgrounds, and communication styles.
- Protect private information and report security concerns through the private security channel.
- Accept moderation decisions intended to keep project spaces productive and safe.

## Unacceptable behavior

Harassment, discrimination, threats, sexualized attention, deliberate intimidation, doxxing, sustained disruption, and publishing another person's private information are not tolerated.

## Enforcement

Report conduct concerns privately through [GitHub Security Advisories](https://github.com/speadskater/graphward/security/advisories/new). Project maintainers may edit or remove content, close or lock discussions, reject contributions, or temporarily or permanently restrict participation. Reports will be handled as confidentially as practical.

This policy applies in project repositories, issue trackers, reviews, discussions, and other spaces where someone represents the Graphward project.
