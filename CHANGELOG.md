# Changelog

All notable user-facing changes are documented here. Graphward follows semantic versioning while it is in public alpha.

## [0.10.1] - 2026-07-22

### Added

- One-command Codex and Claude Code setup with selectable clients, dry runs, and explicit replacement.
- A bundled `graphward-first` workflow installed into each selected agent's user-level skill directory.
- Local MCP usage and explicitly labeled full-file-equivalent compression reporting without storing prompts, arguments, source, or responses.
- User-scoped background service, shared database, multi-repository dashboard, and watcher restoration.

### Changed

- Renamed the pre-public project and every runtime identity from Localtrace to Graphward.
- Prepared the package, documentation, policies, templates, and CI for public development.
- Canonicalized repository paths across Windows short/long path forms and made platform-specific setup tests portable.
