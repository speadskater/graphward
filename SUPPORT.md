# Support

Graphward is a public alpha maintained on a best-effort basis.

- Use [GitHub Issues](https://github.com/speadskater/graphward/issues) for reproducible bugs and focused feature requests.
- Include your Graphward version, Node.js version, operating system, command, expected result, and sanitized output.
- Run `graphward doctor` and include relevant non-sensitive diagnostics when possible.
- Use [GitHub Security Advisories](https://github.com/speadskater/graphward/security/advisories/new) for vulnerabilities.

Please do not post source code, repository paths, database files, credentials, or private Graphward output from a confidential project unless you have permission to disclose it.
